# ZUJ Course Development Hub

A unified home for three courses developed at ZUJ: **IoT**, **Robotics**, and **Computer Vision**. Each course is maintained in its own repository and surfaced here as part of one site.

<div class="grid cards" markdown>

    Sensors, connectivity, edge processing, and connected-device architecture.

    [Open the IoT course →](iot/index.md)

-   ### Robotics

    Kinematics, control, perception, and autonomous behavior.

    [Open the Robotics course →](robotics/index.md)

-   ### Computer Vision

    Image processing, feature extraction, and modern deep-learning approaches.

    [Open the Computer Vision course →](computer-vision/index.md)

</div>
